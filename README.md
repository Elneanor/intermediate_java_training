《Java 中级编程》
=======

一门面向企业级开发培训的，开放的，Java中级编程课程。

### 基本信息

- **教程讲师**: archcentric
- **教程简介**: 一门面向企业级开发培训的Java中级编程课程，面向中级Java程序员。

todo

### 课程目标
- 分析问题，定义需求，设计和开发解决方案
- 掌握Java面向对象编程技术（抽象、继承、多态、封装和泛型）
- 掌握集成开发、调试和单元测试技术
